### OPCIÓN 2: Protección como Archivo Separado

Si prefieres mantener la protección en archivo aparte:

**1. Coloca `protection.js` en:**
```
dental-portfolio/
└── js/
    └── protection.js
```

**2. Agrega en `index.html` y `patient.html` ANTES de `</body>`:**
```html
<script src="js/protection.js"></script>
</body>
```

## ⚙️ DESACTIVAR PROTECCIONES (Opcional):

Si quieres desactivar alguna protección, busca al final del archivo HTML:

```javascript
// Cambiar true a false para desactivar
enableScreenshotProtection: true  // Capturas
enableRightClickBlock: true       // Clic derecho
enableImageProtection: true       // Marca de agua
```
