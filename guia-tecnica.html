Proceso

Pasos para un caso perfecto

1

Escanea la preparación con el antagonista

El antagonista es obligatorio — sin él no podemos diseñar los contactos oclusales correctamente. Escanea también el registro de mordida en máxima intercuspidación.

Tip: Aísla el campo con rollos de algodón 2 minutos antes de escanear para reducir saliva y mejorar la malla.
2

Verifica la calidad de la malla antes de exportar

Revisa en el software que la zona cervical de la preparación esté completa — sin huecos ni zonas en rojo. Esa zona define el margen de la restauración.

El 23% de los casos que recibimos tienen defectos en el margen cervical. Esto genera un retraso de 2-4 horas mientras te contactamos para reescanear.
3

Exporta con los parámetros correctos

Cada software tiene su menú de exportación. Los parámetros clave son la resolución de la malla y el formato. Ve a la sección de tu software abajo.

Resolución mínima requerida: 0.05 mm en zona cervical. Resolución recomendada: 0.02–0.05 mm.
4

Envía el archivo con la prescripción del caso

Incluye: material deseado (zirconia monocapa / multicapa / e.max / PMMA), color VITA, número de piezas, fecha de entrega requerida.

Exportar desde Exocad DentalCAD

Exocad permite exportar el escáner directamente como STL desde el menú de apertura de caso.

  1. Archivo → Exportar modelo → Seleccionar: STL (ASCII) o STL (Binary)
  2. Resolución: 0.05 mm o mejor
  3. Exportar el maxilar superior, inferior y registro de mordida por separado
  4. Alternativamente: enviar el .constructionfile completo — preferimos esto

Exportar desde 3Shape Trios

  1. Desde 3Shape Communicate: compartir directamente a PRODIGY (solicítanos el enlace de conexión)
  2. Exportación manual: Trios → Export → STL o PLY (con color)
  3. PLY es preferible para casos estéticos — conserva la información de textura del esmalte
Opción recomendada: usar 3Shape Communicate → el archivo llega directamente a nuestro sistema sin perder metadatos.

Exportar desde Medit i700

  1. Medit Link → seleccionar el caso → Export
  2. Formato: STL — seleccionar arcada superior, inferior y mordida
  3. Calidad: seleccionar High o Ultra
  4. También puedes compartir directo desde Medit Link → Add Collaborator

Exportar desde iTero Element

  1. Portal iTero → seleccionar scan → Export STL
  2. El formato nativo .itero también es aceptado — nos lo envías directamente
  3. Resolución: la exportación nativa iTero ya viene optimizada
⚠ Si usas Invisalign ClinCheck, el STL exportado puede tener restricciones de Align Technology. Para casos restauradores usa la exportación STL estándar.

Exportar desde Cerec (Dentsply Sirona)

  1. Cerec SW → caso → File → Export → STL
  2. Exportar el bloque de datos completo: preparación + antagonista + mordida
  3. Resolución estándar de Cerec es suficiente (0.05 mm pre-configurado)
Antes de enviar

Checklist del caso perfecto

  • Archivo STL incluye la preparación completa (sin zonas rojas ni huecos)
  • Antagonista incluido (arcada opuesta completa)
  • Registro de mordida en máxima intercuspidación
  • Indicación del material: zirconia monocapa / multicapa / disilicato / PMMA
  • Color VITA especificado (ej: A2, B1, A3.5)
  • Número de piezas y pieza(s) a restaurar
  • Fecha de entrega requerida
  • NO enviar archivos sin antagonista
  • NO enviar STL con resolución < 0.1 mm en zona cervical
  • NO comprimir en ZIP si no es necesario (aumenta tiempo de revisión)
